# 機能仕様書

## システム①：事前準備サイト

### 概要
リベシティの宿題リストを参考にしたスタイル。
参加者がイベント当日までに自分のペースで進める。**順番は問わない。**

### 画面フロー

```
ログイン（Googleアカウント）
    ↓
OS選択（Mac / Windows）※後から変更可
    ↓
宿題リスト画面（必須・任意に分類）
    ↓
各項目をクリック → 説明・記事・動画・外部リンクへ
    ↓
確認できたらチェックを入れる
    ↓
全必須項目チェック完了 → 「準備完了！」
```

### チェックリスト：共通項目（Mac・Win両方）

| # | 項目 | 種別 |
|---|------|------|
| 1 | WiFi確認（Google speed test） | 必須 |
| 2 | 電源ケーブルの用意 | 必須 |
| 3 | ショートメッセージ（SMS）確認 | 必須 |
| 4 | クレジットカードの用意 | 必須 |
| 5 | Googleアカウント（Gmail）確認 | 必須 |
| 6 | 名札の用意 | 必須 |
| 7 | Claudeアカウントの有無 | 必須 |
| 8 | Claudeの契約プラン確認 | 必須 |

### チェックリスト：Mac固有

| # | 項目 | 種別 |
|---|------|------|
| 1 | macOSバージョンをスクショしてChatGPTに確認 | 必須 |
| 2 | スクショの撮り方（Mac） | 必須 |

### チェックリスト：Windows固有

| # | 項目 | 種別 |
|---|------|------|
| 1 | CPU種類（Intel/ARM）をスクショしてChatGPTに確認 | 必須 |
| 2 | スクショの撮り方（Windows） | 必須 |

### Claudeプランと当日の開始地点

| プラン | 当日の開始ステップ |
|--------|----------------|
| なし | アカウント作成 → Pro登録 → インストール |
| フリー | Pro登録 → インストール |
| Pro以上・未インストール | インストールから開始 |
| Pro以上・インストール済み | 動作確認・体験から開始 |

### リセット機能

- ユーザー自身が「最初からやり直す」ボタンでリセット可能
- OS選択・チェックリスト進捗がリセットされる
- アカウント・プロフィール・画像はリセットされない
- 主催者も管理画面から個別ユーザーをリセット可能

## システム②：当日ディスプレイ

### ステータス定義

| ステータス | フェーズ | 説明 |
|----------|--------|------|
| `not_started` | 事前 | 宿題未着手 |
| `checklist_done` | 事前 | 宿題完了・準備OK |
| `pre_completed` | 事前 | チャットでインストールまで完了 |
| `waiting` | 当日 | 会場到着・未開始 |
| `warming_up` | 当日 | 準備体操中 |
| `installing` | 当日 | インストール中 |
| `token_learning` | 当日 | トークン消費を理解中 |
| `web_vs_app` | 当日 | Web版とアプリ版の違いを体験中 |
| `chat_vs_agent` | 当日 | チャット型とエージェント型を理解中 |
| `experiencing` | 当日 | アイディア・Chrome体験中 |
| `completed` | 当日 | 全て完了 |

### 進捗ステップ（当日）

| ステップ | 進捗 | 内容 |
|--------|------|------|
| 待機中 | 0% | 会場到着 |
| 準備体操 | 15% | スクショ・ドラッグ＆ドロップ・右クリック練習 |
| インストール | 30% | DMGインストール・ログイン・SMS認証 |
| トークン理解 | 50% | トークンの消費を実際に見て理解する |
| Web版 vs アプリ版 | 65% | 違いを体験・理解する |
| チャット vs エージェント | 80% | AIの種類の違いを理解する |
| フォルダ整理・Chrome体験 | 90% | エージェントとして実際に使う |
| アイディア体験 | 100% | 自分の興味で「アイディア」を試す → 完了 |

### ヘルプシステム

- 参加者がカードをタップ → ヘルプ要求ON
- ヘルプキューに自動登録・順番割当
- サポーターが対応 → ✓ボタンで完了 → 次の人が繰り上がり

### ソート・フィルター

| 機能 | 選択肢 |
|------|------|
| ソート | デフォルト / 進捗低→高 / 進捗高→低 / 名前A→Z |
| フィルター | すべて / 進行中 / ヘルプのみ / 完了済み |

## 共通基盤（Firebase）

| 機能 | 用途 |
|------|------|
| Hosting | Webアプリ本体のデプロイ・配信 |
| Authentication | Googleアカウントでのログイン |
| Firestore | 事前準備・当日進捗データのリアルタイム同期 |
| Storage | プロフィール画像（PNG）のアップロード・管理 |

### Firestoreデータ構造

```
users/
  {userId}/
    ├── profile
    │     name, os, profileImageUrl, createdAt
    ├── checklist
    │     wifi, power_cable, sms, credit_card,
    │     google_account, name_badge,
    │     claude_account, claude_plan,
    │     os_update（Macのみ）, mac_screenshot（Macのみ）
    │     win_type（Winのみ）, win_screenshot（Winのみ）
    └── event_progress
          status, progress, helpOrder
```
