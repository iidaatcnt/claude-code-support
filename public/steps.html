インストール当日手順

上から順に進めてください。完了したら右端の〇をタップ！

0準備
事前チェック
ノートPC・Wi-Fi・スマホを確認

確認リスト

  • ノートPC（タブレット不可）を開いている
  • Wi-Fiに接続できている
  • スマホを横に置いている（SMS認証用）
  • 名札をつけている
  • OSのソフトウェアアップデートが完了している
💡 白衣のおじさん（サバ缶）を見つけたら声をかけてください！
1DL
デスクトップ版をダウンロード
公式サイトからクロコをダウンロード

公式サイト

https://claude.com/download


Mac
  • 選択肢は1つだけ → そのままダウンロード

Windows
  • 自分のPCに合った版を選んでダウンロード
  • IntelかARMか分からない場合：
    「設定 → システム → バージョン情報 → デバイスの仕様」のスクショをAIに質問
💡 確認ツール：事前確認アプリ → 「msinfo32」で調べられます
2設置
デスクトップ版をインストール
ダウンロードしたファイルを実行
Mac
  • ダウンロードした .dmg ファイルを開く
  • クロコアイコンを「アプリケーション」フォルダへドラッグ＆ドロップ
Windows
  • ダウンロードした Claude.exe をダブルクリック
  • 画面の指示に従うだけで完了
💡 インストール後はダウンロードフォルダのファイルを削除してOK
3Git
Git をインストール
バージョン管理ツールを入れる
Mac

ターミナルを起動

⌘（Command）+ スペース → 「ターミナル」と入力 → Enter

コマンドを実行

xcode-select --install
  • ポップアップが出たら「インストール」→「同意する」
  • 数分かかります。完了まで待ちましょう

確認コマンド

git --version

バージョン番号が表示されれば成功！

Windows

公式サイトからダウンロード

gitforwindows.org

  • 「Download」ボタンをクリック
  • .exe ファイルを実行（設定はすべてデフォルトでOK）

確認方法

コマンドプロンプト または PowerShell を開いて：

git --version

バージョン番号が表示されれば成功！

注意ポイント

手順

  • インストールしたClaudeアプリを起動
  • 「Anthropicアカウントでログイン」または「Googleでログイン」

  • プランは無料プランでOK（後から変更可能）
💡 ここで詰まったらサバ缶を呼んでください。人によって環境が違います！
