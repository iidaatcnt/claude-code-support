クロコとは？
🤖
エージェント型AI
従来のチャット型AIとは違い、あなたの指示を受けて自律的に作業を進めます。「これをやって」と頼むだけでOK。
💻
デスクトップアプリ版
Claude Code のデスクトップアプリ版です。ファイル操作・コード生成・検索など、PCの作業を代わりにこなします。
🏭
産業革命レベルの変化
「AIに仕事を頼む」時代が来ました。使いこなせる人と使えない人で、圧倒的な差が生まれる時代です。
WHY KUROKO?

あなたの働き方が、
今日から変わる。

クロコは、プログラミングの知識がなくても使えます。
「このデータをグラフにして」「メールの文章を直して」
日本語で頼むだけで、クロコが代わりに動きます。

このオフ会では、クロコを使いこなすための
最初の一歩を、一緒に踏み出します。
